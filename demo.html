Architecture

Data flow & design

16 files, ~4,000 lines. Zero external dependencies. Pure SwiftUI + AppKit.

Raw text (paste) | v LogParser.parse() Concurrent parsing (all CPU cores) | Detects: CRI -> Loki -> Zap -> JSON -> plain | Reconstructs CRI P/F chunks in single pass v [LogEntry] Stored in LogViewModel.entries | +---> LogListView Filtered display, level badges, diff marking | | Arrow keys for navigation | v | PayloadCache LRU (20 entries), background pre-warming | | Task.detached (cancellable) | v | LogDetailView Tree / JSON / Raw / Table (switch, not ZStack) | Single @State Built struct, no duplicate copies | +---> DiffView (on demand) | +-- JSONFormatter.prettyPrint() x 2 +-- LineDiffer.diff() Myers O(ND), sparse history, max 20K edits +-- DiffAttrBuilder.build() x 2 NSAttributedString with diff colors +-- Side-by-side NSTextView Scroll sync, Cmd+F search | On close: clear textStorage -> nil contentVC -> nil window -> malloc_zone_pressure_relief PerformanceMonitor phys_footprint (Activity Monitor metric) + RSS task_thread_times_info for CPU % Samples every 2s, ~1 microsecond per sample
Features

Everything you need, nothing you don't

Zero external dependencies. One lightweight binary. Lives in your menu bar.

⌨

Global Hotkey

Ctrl+Shift+Space summons Leif from any app. Never leaves the menu bar.

⚡

Concurrent Parsing

Parses across all CPU cores. 692 lines in 1.6ms. CRI P/F reconstruction in single pass.

🔍

8 Log Formats

CRI, Loki 2-line, Zap dev, JSON object/array, pretty-printed JSON, Go structs, plain text.

🌳

JSON Tree View

Collapsible key/value tree. String-encoded JSON auto-expanded. Search highlights inline.

📊

Table Extraction

Arrays of objects auto-detected. Sortable table. TSV copy + CSV download.

↔

Myers Diff

Same algorithm as git. Side-by-side with scroll sync. Sparse history for memory efficiency.

⏰

Unix Converter

Seconds or milliseconds to UTC ISO 8601. Multi-row with notes. Tab/Enter keyboard flow.

📊

Performance Monitor

Real-time memory (phys_footprint + RSS) and CPU in toolbar. Click to expand details.

🎨

Dark / Light Mode

Toggle instantly. Preference persisted across sessions. Both modes fully styled.
